# Proposal: PowerPlaza × Volker Power 국내 판매 사이트

## Change ID
`powerplaza-site`

## Problem Statement
파워프라자(PowerPlaza)는 Volker Power의 인버터 제품군을 국내에 소개하고 판매하기 위한 전용 웹사이트가 필요합니다. 현재 Volker Power 공식 사이트는 영문으로만 제공되며, 국내 B2B 고객(산업용 전원 담당자, 구매 담당자 등)이 제품 정보를 한국어로 확인하고 견적을 문의할 수 있는 창구가 없습니다.

## Proposed Solution
Next.js 15 기반의 한국어 제품 소개 및 견적 문의 사이트를 구축합니다.

주요 기능:
1. **제품 카탈로그** — Volker Power 인버터 제품(Power Inverter, UPS Inverter, PCBA 등)을 카테고리별로 한국어로 소개
2. **제품 상세 페이지** — 사양(입출력 전압/전류, 정격출력, 인증 등), 이미지, 특징 설명
3. **견적 문의 폼** — 제품별 견적 요청 폼 (이름, 회사명, 연락처, 수량, 요청사항)
4. **회사 소개 페이지** — PowerPlaza 회사 소개, 연혁, 파트너십
5. **관리자 페이지** — 제품 등록/수정/삭제, 문의 내역 확인

## Scope

### In Scope
- Next.js 15 + tRPC + Drizzle ORM + SQLite 기반 풀스택 앱
- 제품 목록 페이지 (카테고리 사이드바 + 그리드)
- 제품 상세 페이지 (이미지, 사양표, 견적 문의 버튼)
- 견적 문의 폼 (DB 저장, 이메일 알림 없이 DB 저장만)
- 회사 소개 페이지 (About Us)
- 관리자 인증 (better-auth) + 관리자 대시보드
- 관리자 제품 CRUD
- 관리자 문의 목록 조회
- 반응형 디자인 (모바일/태블릿/데스크톱)
- 한국어 인터페이스

### Out of Scope
- 실제 결제 시스템 (이커머스 카트/결제)
- 이메일 알림 발송
- 다국어(영문) 지원 (V1에서는 한국어만)
- 실시간 재고 연동
- Volker Power API 연동

## Success Criteria
- [ ] 제품 목록이 카테고리별로 표시되고, 사양이 카드에서 직접 보인다
- [ ] 제품 상세 페이지에서 전체 사양표와 인증 정보를 확인할 수 있다
- [ ] 견적 문의 폼 제출 시 DB에 저장되고 사용자에게 확인 메시지가 표시된다
- [ ] 관리자가 로그인 후 제품을 추가/수정/삭제할 수 있다
- [ ] 관리자가 문의 목록을 조회할 수 있다
- [ ] 모바일에서 레이아웃이 깨지지 않는다
- [ ] Lighthouse 성능 점수 80점 이상

## Key Design Decisions
- **디자인**: 산업용 B2B 분위기 (네이비 헤더, 앰버 CTA, 그레이 캔버스)
- **타이포그래피**: Pretendard (한국어) + IBM Plex Mono (사양값)
- **제품 카드**: 핵심 사양 4개를 카드에서 바로 표시 (클릭 없이)
- **이미지**: Volker Power CDN 이미지 URL 사용 (외부 링크)
- **관리자**: better-auth로 세션 관리, 별도 /admin 경로
